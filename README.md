# go-config-runtime
Realtime configuration updates on the fly (runtime). No need to restart the application.


### Config
| K | V
| ------ | ------ | 
| Config File | Json
| Loader | Viper
| Listener | Viper 


#
### Config Name
- mode
- user
- password
- token
